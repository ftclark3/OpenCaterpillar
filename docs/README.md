This repository depends heavily on openawsem, which is available at
https://github.com/cabb99/openawsem.

We add a few extra forces to the `functionTerms` folder 
for Ivan Coluzza's [original Caterpillar model](https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0020853) and related work.

We also refactor the `System` setup to allow for alternative
backbone models.
